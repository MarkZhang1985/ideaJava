1.在JAVA中，如何跳出当前的多重嵌套循环？

2.switch是否能作用在byte上，是否能作用在long上，是否能作用在String上？


写出结果。
public class Demo{ 
	public static void main(String []args){ 
		int i = 0, j = 5; 
		tp: for (;;){ 
			i++; 
			for(;;){
				if(i > j--)
					break tp; 
			}
		} 
		System.out.println("i = " + i + ", j = "+ j); //i=1,j=-1;
	} 
}
